# Solaleh Kingdom

This is a simple static site that can be hosted for free.

## Best free option

### GitHub Pages

1. Create a new GitHub repository.
2. Push this folder to that repository.
3. In the repo settings, open **Pages**.
4. Set the source to the `main` branch and the root folder.
5. Wait a minute or two, then open the published URL on your phone and laptop.

## Another easy option

### Netlify

1. Create a free Netlify account.
2. Drag and drop this folder into Netlify, or connect the GitHub repo.
3. Netlify will give you a public link that works on any device.
4. The included `netlify.toml` publishes the site from the project root.

## What this site already supports

- Mobile-friendly layout
- Works as a static site
- No server framework required

## Recommended choice

If you want the easiest "works everywhere" setup, use Netlify. If you already use GitHub, use GitHub Pages.
